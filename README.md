# SQLWhisperer 🎯

SQLWhisperer is an elegant natural language to SQL query converter powered by LangChain and Groq. It enables users to interact with databases using plain English, making database querying accessible to everyone - from business analysts to product managers.

## ✨ Features

- Natural language to SQL conversion
- Interactive chat interface
- Support for custom database schemas
- Context-aware query generation
- Real-time SQL explanation
- Vector-based schema understanding
- Multi-file support for schemas and example queries

## 🚀 Key Applications

- **Business Intelligence**: Help analysts query databases without deep SQL knowledge
- **Database Education**: Learn SQL through natural language interactions
- **Rapid Prototyping**: Quickly test database queries during development
- **Data Exploration**: Simplify database exploration for non-technical stakeholders
- **Query Validation**: Get explanations for generated SQL queries to ensure accuracy

## 🛠️ Technologies Used

- Streamlit for the web interface
- LangChain for LLM orchestration
- Groq for language processing
- FAISS for vector storage
- HuggingFace for embeddings
- Python 3.x

## 💡 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your Groq API key
4. Run the application:
   ```bash
   streamlit run app.py
   ```
